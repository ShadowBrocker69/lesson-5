Типы селекторов

По названию элемента:
Данный селектор применяется ко всем элементам с определённым тегом.

Например: h1 {color: blue;}

По id:
Сначала надо присвоить элементу id после чего обратиться в css к этому id и задать ему необходимы параметры, но использовать один id можно лишь единожды

Например: #title {color: aqua;}

Обращение к id: id="title"

По классу:
Аналогичено id, но можно применять сколько угодно раз

Например: .text {color: green;}

Обращение к классу: class="text"

По нескольким классам:
Позволяет "совместить" несколько зараенее созданных классов в один и присвоить этому селектору определённые параметры. Т.е, если в тег вписать оба класса, то используются параметры из этой комбинациию

Например: .text {color: green;} .text2 {color: gray}

Обращение к классу: class="text text2"

По нескольким id:
Аналогичено нескольким классам
По классу и id:
Аналогично нескольким классам, но в данном случае совмещаются и класс, и id

Например: #title {color: aqua;} .text {color: red;}

Обращение к классу + id: class="text" id="text"

По вложенности:

Например: .text {color: green;}

Обращение к классу: class="text"
